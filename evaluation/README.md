# Evaluation

本项目使用自动指标与用户评测结合的方式验证古诗文生成质量，覆盖语义保真、文本相似度、语言流畅度和主观可用性。

## 评测体系

| 评测维度 | 指标 | 作用 |
|---|---|---|
| 文本重合度 | BLEU | 衡量生成文本与参考古诗文的 n-gram 匹配程度 |
| 序列相似度 | ROUGE-L | 衡量最长公共子序列，反映结构与表达接近度 |
| 语言建模质量 | Perplexity | 衡量模型对目标文本分布的拟合程度 |
| 基线对比 | RNN / LSTM | 对比传统序列生成模型与 LLM 微调方案 |
| 用户体验 | 12 名用户评测 | 从古风程度、语义保真、可读性进行主观判断 |

## 核心结果

| 指标 | 微调前 | LoRA 微调后 | 提升 |
|---|---:|---:|---:|
| BLEU | 0.45 | 0.60 | +33.3% |
| ROUGE-L | 0.38 | 0.55 | +44.7% |
| Perplexity | 110 | 85 | -22.7% |

## 文件说明

- `eval_results.csv`：模型生成样例、参考文本与 Perplexity 结果。
- `loss_curve.png`：训练过程损失曲线。

这些结果用于展示模型迭代效果和产品评测闭环。
